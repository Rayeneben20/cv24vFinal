Liste des Opérations

URL Méthode Résumé Reçoit Retourne
/ GET Affiche la page d'accueil - text/html
/help GET Affiche la page d'aide - text/html
/cv24.xsd GET Télécharge le fichier XSD cv24 - application/xml
/cv24/resume/xml GET Retourne le résumé de tous les CVs en format XML - application/xml
/cv24/resume/html GET Retourne le résumé de tous les CVs en format HTML - text/html
/cv24/insert POST Insère un nouveau CV valide dans la base de données application/xml application/xml
/cv24/delete DELETE Supprime le CV avec l'identifiant spécifié dans le corps de la requête text/plain application/xml
/cv24/search?date=‹date›&objectif=‹objectif› GET Retourne le résumé de tous les CVs en format XML répondant aux critères :
  • dates égales ou postérieures à ‹date> (RFC3339 YYYY-MM-DD)
  • et au moins une correspondance de l'objectif dans la liste de mots de ‹objectif›
(au moins un paramètre est requis, ordre des paramètres non pertinent) 		- application/xml
/cv24/xml?id=‹id› GET Retourne le CV avec l'identifiant spécifié en format XML - application/xml
/cv24/html?id=‹id› GET Retourne le CV avec l'identifiant spécifié en format HTML - text/html
